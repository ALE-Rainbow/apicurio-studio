Why Am I Seeing/Doing This?

When you click the Authenticate Now button above, you'll be taken to GitHub and (perhaps) presented with a screen asking you to authorize the Apicurio Studio application to be able to perform some functions on your behalf. This is needed so that the Studio can do things like commit OpenAPI definitions directly into your GitHub repositories. We also do a few things like display a list of your GitHub organizations on a couple of screens.

I Don't Have a GitHub Account!

Don't worry about it! When you click the Authenticate Now button above, you will be taken directly to GitHub where, if you don't already have an account, you can create one!
